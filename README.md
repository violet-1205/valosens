# Valosens

Valosens는 **Valorant 플레이어의 마우스 감도(Sensitivity)를 테스트 기반으로 점검하고 추천값을 제안하는 웹 시뮬레이션 서비스**입니다.

감도를 감으로 맞추는 대신, 발로란트의 실제 플레이 방식(멈추고 쏘기)에 맞춘 3가지 테스트에서 수집한 데이터를 기반으로 결과 리포트를 제공하고 추천 감도 출발점을 제시합니다.

## 서비스 소개

### 감도 설정

테스트 시작 전 현재 사용 중인 **DPI**와 **발로란트 인게임 감도**를 입력합니다.
입력한 값으로 eDPI와 cm/360°가 실시간 계산되어 표시되며, 세 테스트 모두 이 감도 기준으로 진행됩니다.

### 3단계 테스트

- **Test 1 · 360° 회전 정밀도:** 마커를 배치한 뒤 360° 회전 후 정확히 원점에 돌아오는 능력을 측정합니다. 마우스 이동량(px)과 각도 편차(°)를 기록해 감도 제어력을 파악합니다.
- **Test 2 · 코너 플릭킹:** 머리 높이에 고정된 타겟이 좌우 교대로 등장합니다. 발로란트의 코너 피킹 상황을 시뮬레이션하며, 오버슈트/언더슈트 경향을 분석합니다.
- **Test 3 · 탭샷:** 20개의 정지 타겟이 1.5초 제한으로 순차 등장합니다. 발로란트의 멈추고 쏘는 에임 방식에 맞게 탭샷 정확도(%)와 평균 반응속도(ms)를 측정합니다.

### 결과 리포트

3개 테스트 데이터와 입력한 현재 감도를 종합하여 **현재 감도 vs 추천 감도**를 나란히 비교합니다. 변화량(+/-)과 조정 방향 설명을 함께 제공합니다.

## 사용자 진행 흐름

1. 홈 화면에서 **감도 측정 시뮬레이션 시작** 클릭
2. 감도 설정 모달에서 DPI + 발로란트 인게임 감도 입력 → 확인
3. Test 1 → Test 2 → Test 3 순서대로 진행
4. 결과 화면에서 현재/추천 감도 비교 및 조정 방향 확인
5. 필요 시 재시작하여 반복 측정/비교

각 테스트의 설정값과 결과는 브라우저 `localStorage`에 저장됩니다.

## 핵심 기능

- **감도 설정 모달:** DPI 프리셋(400/800/1600/3200), 인게임 감도 입력, 실시간 eDPI · cm/360° 미리보기
- **발로란트 특화 시뮬레이션:** 오버워치 스타일의 트래킹 대신 코너 플릭과 탭샷 중심으로 설계
- **360° 정밀도 측정:** 각도 편차(°)로 감도 제어 안정성 정량화
- **현재 감도 기반 추천:** 고정 기준값이 아닌 실제 사용 감도에서 상대 조정
- **현재 vs 추천 비교 UI:** 변화량 뱃지와 조정 방향 메시지 포함
- 라이트/다크/시스템 테마 지원
- FPS 느낌의 3D 인터랙티브 시뮬레이션 (Pointer Lock API)

## 기술 스택

- **Frontend:** React 18, React Router v7
- **3D:** Three.js, @react-three/fiber, @react-three/drei
- **Styling:** Tailwind CSS v4
- **Build Tool:** Vite 6

## 프로젝트 구조

```
src/
├── App.jsx                   # 라우팅 (/,  /test1, /test2, /test3, /result)
├── pages/
│   ├── Home.jsx              # 서비스 소개 + 감도 설정 모달
│   ├── Test1.jsx             # 360° 회전 정밀도 테스트 페이지
│   ├── Test2.jsx             # 코너 플릭킹 테스트 페이지
│   ├── Test3.jsx             # 탭샷 테스트 페이지
│   └── Result.jsx            # 결과 리포트 + 감도 추천 비교
└── components/
    ├── RotationSim.jsx       # 360° 회전 시뮬레이션 (각도 편차 측정 포함)
    ├── FlickingSim.jsx       # 코너 플릭킹 시뮬레이션 (머리 높이 고정, 좌우 교대)
    ├── TrackingSim.jsx       # 탭샷 시뮬레이션 (20개 정지 타겟, 1.5초 제한)
    └── Layout.jsx            # 공통 레이아웃 (헤더/푸터/테마)
```

## 참고

추천 감도는 테스트 결과 기반의 시작점입니다.
실제 게임에서는 역할, 맵, 플레이 스타일에 맞춰 미세 조정을 권장합니다.
